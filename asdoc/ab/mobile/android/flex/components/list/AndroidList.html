Packageab.mobile.android.flex.components.list
Classpublic class AndroidList
InheritanceAndroidList spark.components.List

The AndroidList class extends the Spark List component to provide the capability to bring up an options menu pop up on touch and hold in the same style as the Android native list component.

enableItemOptionsproperty
enableItemOptions:Boolean

When set to true and an itemOptions Array has been set the List will prompt a set of options to choose from. Choosing an option will dispatch a ListEvent.OPTION_CHOSEN event with an item property and the option label selected.
